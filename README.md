# multi-convertor
Multi Converter is a simple and user-friendly web application that provides multiple useful conversion tools in one place. The project is built using HTML, CSS, and JavaScript and is designed to make everyday calculations quick and easy.

This project includes three main tools:

Currency Converter 💱 – Convert values between different world currencies quickly.

Unit Converter 📏 – Convert units such as length, weight, temperature, and more.

Age Calculator 📅 – Calculate your exact age in years, months, days, hours, minutes, and seconds.

The goal of this project is to create a clean, responsive, and easy-to-use interface where users can access multiple conversion tools without needing different websites.

Features

Simple and intuitive user interface

Multiple converters in one platform

Fast calculations using JavaScript

Responsive design for desktop and mobile

Lightweight and easy to use

Technologies Used

HTML – Structure of the website

CSS – Styling and layout

JavaScript – Calculation logic and interactivity

Project Purpose

This project was created to practice frontend web development and demonstrate how multiple utilities can be combined into a single web application.
